# AI 블로그 생성 사이트 - 제품 비전 및 기능 정의

작성일: 2026-03-04
담당: 기획 에이전트 1

---

## 1. 제품 비전

누구나 주제 하나만 입력하면, AI가 전문적인 아티클·이미지·참고자료를 자동으로 생성해주는 블로그 콘텐츠 플랫폼을 제공한다. 콘텐츠 생산의 진입 장벽을 낮추고, 개인과 팀이 고품질 블로그를 빠르게 발행할 수 있도록 돕는다.

---

## 2. 타겟 사용자 (3가지 페르소나)

### 페르소나 A — 1인 창업자 / 스타트업 마케터 "지수"
- 나이: 28세
- 상황: 제품 홍보를 위한 블로그 콘텐츠가 필요하지만 글쓰기에 시간을 쓸 여유가 없음
- 니즈: 주제 키워드만 넣으면 SEO 친화적인 초안이 빠르게 나왔으면 좋겠다
- 페인 포인트: 외주 작가 비용 부담, 긴 피드백 사이클

### 페르소나 B — 테크 블로거 / 사이드 프로젝트 개발자 "민준"
- 나이: 34세
- 상황: 개인 기술 블로그를 운영하지만 꾸준한 포스팅이 어렵고 초안 작성에 많은 시간을 소모
- 니즈: 기술 주제에 맞는 구조화된 글과 관련 이미지가 자동으로 생성되길 원함
- 페인 포인트: 초안 작성 시간 과다, 적절한 이미지 찾기 번거로움

### 페르소나 C — 중소기업 콘텐츠 팀 매니저 "소연"
- 나이: 41세
- 상황: 팀원 2~3명으로 월 20건 이상의 블로그 발행 목표를 달성해야 함
- 니즈: 빠른 초안 생성 후 팀원이 편집만 해서 발행하는 워크플로우
- 페인 포인트: 작가별 품질 편차, 반복 작업으로 인한 번아웃

## 3. 핵심 기능 목록 (MoSCoW 우선순위)

### Must Have (MVP — 반드시 구현)
- 주제 키워드 입력 → AI 블로그 아티클 자동 생성 (Claude API)
- 생성된 글의 섹션 구조 (제목, 소제목, 본문, 결론)
- 관련 이미지 자동 삽입 (Unsplash / Pexels API)
- 참고자료 링크 자동 수집 (웹검색 API)
- 생성 결과 미리보기 (읽기 전용)
- 생성된 글 마크다운 / HTML 내보내기

### Should Have (중요하지만 MVP 이후 가능)
- 생성된 글 온라인 편집기 (인라인 수정)
- 글 저장 및 히스토리 관리 (사용자 계정)
- 톤 & 스타일 설정 (격식체/비격식체, 분량 조절)
- SEO 메타태그 자동 생성 (title, description, keywords)
- 다국어 출력 지원 (한국어 / 영어 선택)

### Could Have (여유가 있을 때)
- 소셜 미디어용 요약 카드 자동 생성
- 브랜드 가이드라인 적용 (폰트, 색상 커스터마이징)
- 팀 협업 기능 (댓글, 승인 워크플로우)
- 예약 발행 및 외부 CMS 연동 (WordPress, Notion 등)
- 성과 분석 대시보드 (조회수, 체류 시간)

### Won't Have (현재 범위 외)
- 자체 블로그 호스팅 플랫폼
- 동영상 / 팟캐스트 콘텐츠 생성
- AI 이미지 생성 (저작권 이슈로 스톡 이미지 우선)
- 실시간 다중 사용자 공동 편집

## 4. 경쟁 서비스 차별점 (3가지)

### 차별점 1 — 원스톱 콘텐츠 패키지 생성
텍스트 + 관련 이미지 + 참고자료 링크를 한 번에 묶어 발행 가능한 수준의 초안 제공.

### 차별점 2 — 한국어 최적화 & 로컬 콘텐츠 감수성
Claude API의 한국어 이해도를 활용해 한국 독자에게 자연스러운 문체와 국내 트렌드를 반영.

### 차별점 3 — 투명한 참고자료로 신뢰도 확보
웹검색 API 연동으로 검증 가능한 참고자료 링크를 명시.

## 5. MVP 범위 정의

### In Scope
| 항목 | 설명 |
|---|---|
| 주제 입력 UI | 키워드 + 선택적 추가 지시사항 입력 폼 |
| AI 아티클 생성 | Claude API, 제목~결론까지 구조화된 본문 |
| 이미지 자동 삽입 | Unsplash API, 키워드 기반 1~3장 |
| 참고자료 수집 | 웹검색 API, 관련 링크 3~5개 |
| 결과 미리보기 | 렌더링된 형태로 표시 |
| 내보내기 | 마크다운(.md) 또는 HTML 다운로드 |

### Out of Scope (MVP)
- 회원가입 / 로그인 / 마이페이지
- 글 저장 및 편집 (생성 후 즉시 다운로드로 대체)
- SEO 분석, 성과 추적, 팀/협업 기능

### MVP 성공 기준
- 주제 입력부터 초안 확인까지 60초 이내
- 생성된 글 최소 800단어 이상
- 이미지 1장 이상 자동 삽입 성공률 95% 이상
- 참고자료 링크 3개 이상 제공
